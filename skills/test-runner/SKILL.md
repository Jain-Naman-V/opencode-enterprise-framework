## What I do
- Detect the test framework used in the project
- Run test suites with appropriate flags
- Analyze test results and failures
- Suggest fixes for failing tests
- Identify patterns in test failures

## When to use me
Use this when you need to run tests or analyze test results. Ask clarifying questions if you need specific test scope or particular test files.

## Supported Frameworks
- JavaScript/TypeScript: Jest, Vitest, Mocha
- Python: pytest, unittest
- Rust: cargo test
- Go: go test
- And many more

## Best Practices
- Run tests frequently during development
- Focus on failing tests first
- Look for patterns in failures
- Write tests that fail for the right reasons
